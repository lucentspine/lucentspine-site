## Dr. Lee's Professional Photo

The professional headshot of Dr. Stanley S. Lee needs to be extracted from the CV PDF located at:
`assets/documents/Stanley-Lee-CV.pdf` (Page 1, upper left corner)

The photo shows Dr. Lee in a white medical coat with a professional studio background. This should be:
1. Extracted from the PDF
2. Saved as `dr-lee-photo.jpg`
3. Optimized for web use (recommended size: 400x400 or similar)

### Manual Extraction Steps:
1. Open Stanley-Lee-CV.pdf in a PDF reader
2. Use screenshot or snipping tool to capture the photo (upper left of page 1)
3. Crop to just the photo area
4. Save as `dr-lee-photo.jpg` in this directory
5. Delete the placeholder file
